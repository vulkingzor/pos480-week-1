Program Requirements:

Key parts:
 - The GUI program will have a button that creates a new window with the word Hello.
 - Enhance the display by making the word change color, move, or change to another language (such as Hola).


Program Flow:
- First Form Opens with Title and Say Hello Button.
- Once "Say Hello" is clicked, a second Form opens with Hello Written.
- Buttons for change color and change language are on the second form. These change the text accordingly.

Input: 
- Button inputs changes language or color. 

Output:
- The label of hello that changes as the user pushes the on GUI buttons
